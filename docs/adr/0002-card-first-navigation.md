# 付款码作为默认页面

GSAULife 延续 GSAU-Card 以付款为核心的使用方式，启动后默认显示付款码页面。底部导航由“教务、付款码、设置”三个入口组成，付款码位于中央；中央入口用于返回付款码页面，页面中的大二维码再进入全屏付款。成绩、课表、考试与排名集中在教务入口中，设置入口容纳教务身份状态、应用设置与关于信息。底栏在教务功能页保持可见，在全屏付款页与统一认证页隐藏，使付款码始终能从日常页面一步抵达。离开教务区域后不保留其内部页面，重新选择教务入口时从四宫格开始。系统返回键从教务功能页返回四宫格，从教务或设置根页返回付款码页面，并在付款码根页退出应用。
